On OSX git credentials can be store in the keychain.

git config --global credential.helper osxkeychain

Another method is to store the credentials in the git folder. Beware, the credentials will be stored in plain text.

cd /path/to/repos
git config remote.origin.url https://username:password@github.com/username/repos
