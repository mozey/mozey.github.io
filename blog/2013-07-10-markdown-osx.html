To create markdown on OSX I use this app.

Mou

And there is also this generator for Quicklook (press spacebar when the file is selected in Finder).

qlmarkdown

Download

Extract

unzip QLMarkdown-xxx.zip

Install

mv QLMarkdown.qlgenerator ~/Library/QuickLook

Reset quicklook

qlmanage -r

Or relaunch finder

alt + right click Finder and select Relaunch
