Stop the database service sudo service mysql stop Make sure it is not running ps ax | grep mysql Start the server using this option (taking not of the pid) sudo mysqld --skip-grant-tables & Login as the root user mysql -u root And change the root password update mysql.user set Password=PASSWORD('myNewPassword') WHERE User='root'; flush privileges; exit Stop the database service and restart sudo kill pid sudo service mysql start Login using the new password mysql -u root -p
