Sometimes when scripting we need to allow certain groups or users to run sudo without being prompted for a password.

We might also want limit this privilege to certain files.

sudo visudo

/etc/sudoers must be edited using above command

We can all a group to sudo without a password

%admin ALL=(ALL) ALL

Or a user

ubuntu ALL=(ALL) NOPASSWD:ALL

And we might limit this to certain files

ubuntu ALL=(ALL) NOPASSWD: /path/to/files/*.sh
