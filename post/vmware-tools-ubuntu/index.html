Install VMWare tools in Ubuntu 14.04

Install dependencies

sudo apt-get install build-essential

sudo apt-get install linux-headers-`uname -r`

Run VMWare configuration tools

Select install VMWare tools from Fusion menu

sudo mount /dev/cdrom /media/cdrom

cd ~ && tar -xvf /media/cdrom/VMwareTools-9.9.3-2759765.tar.gz

cd ~/vmware-tools-distrib && sudo ./vmware-install.pl

Install script will run /usr/bin/vmware-config-tools.pl

Mount shared folder

Use the menu to name the share

sudo mkdir -p /mnt/hgfs/SHARE_NAME

sudo vmware-hgfsmounter .host:/`vmware-hgfsclient` /mnt/hgfs/SHARE_NAME

Install Open VM tools in Ubuntu 14.04

NOTE: Don’t install this, just install VMWare tools

VMware Tools in an Ubuntu 14.04 Guest

"Open VM Tools is the open source implementation of VMware Tools...
VMware recommends using the Open VM Tools redistributed by the operating system vendors"

Install Open VM tools

sudo apt-get install open-vm-tools

Add the vmhgfs Driver

"If you are using the interactive file drag-and-drop... install the vmhgfs driver"

Add the deployPkg Tools Plug-in

"If you are using the virtual machine as a template or if it will be protected by SRM (Site Recovery Manager), then install the deployPkg Tools plug-in"

Mount a shard folder

sudo mkdir -p /mnt/hgfs/SHARE_NAME

sudo vmware-hgfsmounter .host:/`vmware-hgfsclient` /mnt/hgfs/SHARE_NAME
