git

List log for a specific path

git log -- path/to/list

git log --oneline -- path/to/list

Roll back to a specific commit

git branch temp

git reset --hard 41508773bb0f816a9fbb4c6f60aa4d784b25fc0e

git branch -D temp

Ignoring files

Create a global .gitignore

git config --global core.excludesfile ~/.gitignore_global

Changing a remote’s URL

List existing remotes

git remote -v

Change remote

git remote set-url origin https://change.url.for.origin.to.this

Stashing

Stash changes

git stash

Working directory should be clean

git status

There can be multiple stashes

git stash list

Apply the most recent stash

git stash apply

Apply a specific stash

git stash apply stash@{2}

Apply stash and re-stage files

git stash apply --index

Remove a stash

git stash drop stash@{0}

Un-apply a stash

git stash show -p stash@{0} | git apply -R

Undo the last commit

Undo the commit, leaving files intact

git reset --soft HEAD~

Undo the commit and everything that was staged, leaving files intact

git reset HEAD^

Reset to previous commit, throwing away changed made in between

git reset --hard HEAD^

Self signed certificate

git config --global http.sslVerify false
