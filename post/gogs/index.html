gogs

A painless self-hosted Git service.

Install

Requires

MySQL or PostgreSQL

Git

SSH server

Download binary and extract

wget https://dl.gogs.io/gogs_v0.9.13_linux_amd64.tar.gz

tar xvzf gogs_v0.9.13_linux_amd64.tar.gz

Create database

cd gogs

mysql -u root -p < scripts/mysql.sql

Create custom config, example

mkdir -p gogs/custom/conf

vi gogs/custom/conf/app.ini

Run server

cd gogs && ./gogs web

Connect via web interface to complete install

Create self signed cert

mkdir custom/https && cd custom/https

../../gogs cert -ca=true -duration=8760h0m0s -host=host.example.com

Update config for HTTPS

PROTOCOL = https
DOMAIN = host.example.com
ROOT_URL = https://host.example.com/
CERT_FILE = custom/https/cert.pem
KEY_FILE = custom/https/key.pem

Usage

Change remote to use https

git remote -v

git remote set-url origin https://git.mozey.co:8080/mozey/crm-project.git

Disable SSL verification for single repository

git config http.sslVerify false

go get

Private repository

cd $GOPATH/src

mkdir /path/as/required && cd /path/as/required

git clone git@github.com:private/repos.git

Update the package

go get -u

Install the package

go install
